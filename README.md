# Đồ án Hệ quản trị CSDL

## Danh sách thành viên
- Lê Hoàng Thiên Ân
- Đào Duy An
- Trần Xuân Sơn

## Cài đặt
Cần sử dụng `yarn`
```bash
git clone --depth 1 --branch main https://github.com/txuanson/hqtcsdl.git
cd hqtcsdl
yarn
```

## Khởi chạy (dev)
```bash
yarn start
```
